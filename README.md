🧳 Travel List App

A simple and interactive Travel List application built with React to manage items you want to pack for your trips.

🔗 Live Demo: https://travel-list-gamma-ten.vercel.app/

🚀 Features
➕ Add new items to your travel list
❌ Delete items easily
✔️ Mark items as packed/unpacked
🔄 Real-time UI updates
📊 Track packing progress
🛠️ Tech Stack
⚛️ React.js
🟨 JavaScript (ES6+)
🎨 CSS
📚 What I Learned

This project helped me understand:

React useState hook for managing state
Handling form inputs and user interactions
Conditional rendering
Rendering lists using map() and keys
Building interactive UI with state-driven logic
🧠 Key Concept

Unlike Vanilla JavaScript, where we manually update the DOM, this app uses React’s state-driven approach, where the UI automatically updates when the state changes.

📦 Getting Started

To run this project locally:

git clone https://github.com/your-username/travel-list.git
cd travel-list
npm install
